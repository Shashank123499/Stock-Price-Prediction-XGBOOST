# Stock-Price-Prediction-XGBOOST
Machine Learning project for stock price prediction using XGBOOST on 5 years of historical stock market data.

# Project Overview

This project predicts stock closing prices using historical stock market data collected over the last five years. The model is built using XGBoost Regressor and includes data preprocessing, feature engineering, model training, evaluation, and visualization.

# Dataset

Dataset: stock_details_5_years.csv<br>
This dataset obtained from kaggle.

# Features include:

Open Price<br>
High Price<br>
Low Price<br>
Volume<br>
Dividends<br>
Stock Splits<br>
Company Name<br>
Date

# Target Variable:

Close Price
